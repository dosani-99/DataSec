1. HTML — 화면의 뼈대
🏗️
HTML (HyperText Markup Language)
화면에 보이는 모든 요소를 만드는 언어
지뢰찾기의 칸, 버튼, 타이머 숫자, MHS 표 같은 것들이 모두 HTML로 만들어졌어요.
꺾쇠괄호(< >)로 감싼 태그를 사용해서 요소를 정의해요.
실제 사용 예시 — 지뢰 칸 1개 만들기:
<!-- 지뢰찾기 칸 하나 -->
<div class="cell"> </div>

<!-- 타이머 LCD 화면 -->
<div class="lcd" id="timer">000</div>

<!-- 검색 버튼 -->
<button onclick="doSearch()">검색 시작</button>
지뢰 칸 81개 생성 MHS 표 구조 버튼 배치 입력창
2. CSS — 화면의 디자인
🎨
CSS (Cascading Style Sheets)
크기, 색깔, 테두리, 배치를 담당하는 언어
Windows XP 느낌의 회색 창, 빨간 LCD 숫자, 입체감 있는 테두리가 모두 CSS예요.
CSS가 없으면 그냥 글자만 나열된 못생긴 화면이에요.
실제 사용 예시 — Windows XP 스타일 적용:
/* 지뢰 칸 스타일 */
.cell {
  width: 26px;
  height: 26px;
  background: #d4d0c8; /* XP 회색 */
  border-color: #fff #808080 #808080 #fff; /* 입체 테두리 */
}

/* LCD 타이머 */
.lcd {
  background: #000;
  color: #f00; /* 빨간 숫자 */
  font-family: 'Courier New';
}
Windows XP 외관 빨간 LCD 숫자 입체 테두리 색상 강조
3. JavaScript — 프로그램의 두뇌
⚡
JavaScript
실제 동작하는 모든 로직을 담당하는 언어
지뢰 배치, 타이머 작동, 칸 클릭 처리, 서브 검색, 값 수정까지
프로그램의 모든 기능이 JavaScript로 작동해요.
HTML·CSS가 아무리 예뻐도 JavaScript 없이는 아무것도 동작하지 않아요.
실제 사용 예시 — 타이머 작동:
// 1초마다 타이머 +1 증가
let gTime = 0;

ticker = setInterval(() => {
  gTime++; // 1씩 증가
  document.getElementById("timer")
    .textContent = gTime; // 화면에 표시
}, 1000); // 1000ms = 1초
타이머 작동 지뢰 랜덤 배치 칸 클릭 처리 서브 검색 로직 메모리 값 수정
세 언어 모두 설치 없이 브라우저만 있으면 바로 실행돼요 — 그래서 수업 자료로 활용하기 좋아요!
